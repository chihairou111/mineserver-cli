# mineserver-cli

An interactive CLI built with Ink to create, manage, and launch local Minecraft server instances.

## Features

- Create server instances and bootstrap configuration
- Choose and install loaders (e.g., Fabric) and manage mods
- Start/stop instances and view status in one place
- Handle common tasks in the terminal without manual config edits

## Install

```bash
npm install -g github:chihairou111/mineserver-cli
```

## Usage

```bash
mineserver
```
